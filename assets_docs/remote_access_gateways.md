# Remote Access Gateways

# Summary
Remote Access Gateways—typically IT-native interfaces like virtual private networks (VPN), SSH, Remote Desktop Protocol (RDP)—are frequently used by third parties to maintain or troubleshoot OT systems on board vessels. This introduces cross-domain trust that can be abused if it is misconfigured or compromised.
