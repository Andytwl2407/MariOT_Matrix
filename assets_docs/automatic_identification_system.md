# Automatic Identification System (AIS)

# Summary
AIS is a shipborne broadcast system used for maritime traffic coordination, collision avoidance, and situational awareness. Due to its lack of authentication and encryption, AIS is vulnerable to spoofing, signal injection, and data manipulation—posing safety and security risks for vessel operations and maritime domain awareness.
