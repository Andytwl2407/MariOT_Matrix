# Shipboard Remote Maintenance Software

# Summary
Shipboard remote maintenance software allows equipment vendors and service providers to perform diagnostics, configuration changes, and firmware updates over remote channels. These platforms often have deep access to OT assets and, if compromised, can serve as direct vectors to deploy malware, manipulate control logic, or bypass the security boundaries of ships.
