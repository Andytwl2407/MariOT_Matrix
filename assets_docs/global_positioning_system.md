# Global Positioning System

# Summary
The Global Positioning System (GPS) is a satellite-based navigation technology that provides real-time position, velocity, and time synchronization. In maritime Operational Technology (OT) environments, GPS is a foundational service enabling precise navigation and coordination of vessel operations. It supports a wide range of onboard systems such as the Automatic Identification System (AIS), Electronic Chart Display and Information System (ECDIS), Autopilot systems, and Vessel Traffic Services (VTS).
