# Emergency Position Indication Radio Beacon 

# Summary
An Emergency Position Indicating Radio Beacon (EPIRB) is a distress signaling device used to transmit a ship's location in case of an emergency at sea. When activated, an EPIRB sends a distress signal via satellite (e.g., COSPAS-SARSAT) to search and rescue services, providing essential location information for rescue operations. EPIRBs are mandatory for most commercial vessels and significantly enhance maritime safety.
