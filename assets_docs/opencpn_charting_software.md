# OpenCPN Charting Software

# Summary
OpenCPN is an open source electronic charting and navigation software that is commonly used on conventionally equipped and autonomous vessels. It provides real-time situational awareness by processing maritime data from onboard sensors, including GPS, radar, and depth sounders. The system supports standard communication protocols such as AIS and National Marine Electronics Association (NMEA) 0183.
