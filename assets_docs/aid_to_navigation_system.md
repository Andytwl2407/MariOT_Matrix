# Aid to Navigation System

# Summary
Aid to Navigation (AtoN) systems are maritime infrastructures designed to support safe vessel navigation, particularly in coastal, harbor, and high-traffic areas. These include physical beacons, buoys, and virtual AtoNs (V-AtoNs), which are broadcasted via AIS without a physical counterpart.
