# Voyage Data Recorder

# Summary
A Voyage Data Recorder (VDR) is a data recording system installed on ships, similar to a "black box" in aircraft. It continuously captures navigational, positional, and sensor data, including voice recordings from the bridge, radar images, AIS (Automatic Identification System) data, and communication logs. The primary purpose of a VDR is to aid in accident investigations, safety assessments, and performance monitoring.
