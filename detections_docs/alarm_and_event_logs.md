# Alarm and event logs

Alarm and event logs record system-generated alerts and operator acknowledgments related to safety, control anomalies, sensor errors, and communication failures. These logs are integral to the vessel monitoring systems and provide insight into how the vessel responds to unexpected behaviors [^1],[^2].
