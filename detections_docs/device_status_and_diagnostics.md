# Device status and diagnostics logs 

Device status and diagnostics logs in maritime OT environments record operational states, internal diagnostics, and performance indicators of shipboard control and monitoring systems. Typical sources include PLC units, HMI terminals, power management modules, and propulsion controllers. These logs include entries related to system restarts, diagnostic fault codes, watchdog timer activations, and transitions between operating modes, including changes from automatic to manual control [^1].
