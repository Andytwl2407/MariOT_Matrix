# Control Logic Changes

Control logic change logs record modifications to automation programs and control sequences implemented on ships’ operating technology devices, including PLC, remote terminal units (RTUs), and propulsion control modules. In maritime contexts, these changes can influence essential operations such as the regulation of engine thrust, ballast system automation, and electrical power distribution [^1]. These logs track events related to the upload, download, editing, and activation of control logic. Modifications can originate from engineering stations on board or remote access sessions established through maintenance interfaces [^2].
