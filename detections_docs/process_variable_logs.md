# Process Variable Logs

Process variable logs in maritime OT environments store time series data generated by sensors and control systems that monitor the physical state of shipboard operations. These logs contain measurements related to engine Revolutions Per Minute (RPM), shaft torque, fuel pressure, ballast tank levels, electrical load and environmental metrics, including temperature, humidity, and CO2 concentration. Data are acquired from PLC devices, data acquisition modules, or the PMS infrastructure[^1].
